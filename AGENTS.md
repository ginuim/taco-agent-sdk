# AGENTS Rules

## README 同步更新规则（强制）

在本项目中，凡是发生以下任一情况，必须在同一次改动中同步更新文档：

- 新增功能
- 修改现有功能行为
- 调整对外 API、配置项、参数或默认值
- 影响用户使用方式、示例代码或集成方式的变更

必须同时更新：

- `README.md`（英文）
- `README.zh-CN.md`（中文）

## 执行要求

- 不允许只更新单语 README。
- 文档内容要与代码行为一致，示例可运行且与当前包名一致。
- 若改动不影响用户侧使用，可在 PR/提交说明中写明 “No README impact” 并给出理由。
